# Entry Template

Use a compact shape that preserves the trade-off.

## Minimal template

```md
## ADR-XXX: Short title

**Date:** YYYY-MM-DD
**Status:** accepted

**Context:** Why this decision exists now.

**Options considered:**
- Option A: upside / downside
- Option B: upside / downside
- Option C: upside / downside

**Decision:** What we chose and why.

**Consequences:** What becomes easier, harder, or newly constrained.

**Revisit if:** What would make this worth revisiting.
```

## Superseding rule

If a prior ADR is replaced:

- mark the old one as superseded
- reference the new ADR explicitly
- do not rewrite history to make the old decision disappear
